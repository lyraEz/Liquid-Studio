# 🌌 Liquid Studio

> **Transforme suas imagens com a estética Liquid Glass e o poder da IA.**

O **Liquid Studio** é um editor de imagens web de alta performance, focado em uma experiência de usuário imersiva e moderna. Desenvolvido com **React** e **Tailwind CSS**, ele entrega efeitos de vidro (glassmorphism) e processamento em tempo real.

## ✨ Funcionalidades

* **🎛️ Edição Profissional:** Ajuste de brilho, contraste, saturação, matiz, desfoque e muito mais em tempo real.
* **🤖 IA Actions:** Ferramentas integradas para remoção de fundo e filtros de estilo (Anime/Artístico).
* **🕒 Histórico Inteligente:** Sistema completo de **Undo** (desfazer) e **Redo** (refazer) para você nunca perder um ajuste.
* **🌗 Antes & Depois:** Comparação rápida para visualizar a evolução da sua edição.
* **📱 Responsivo:** Otimizado para dispositivos móveis e desktop com um design dark/neon premium.

## 🛠️ Tecnologias Usadas

| Tecnologia | Função |
| :--- | :--- |
| **React 19** | Engine da interface |
| **Vite** | Build tool ultra-rápida |
| **Tailwind CSS** | Estilização Liquid Glass & Neon |
| **Lucide React** | Pack de ícones minimalistas |

## 🚀 Como Rodar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/lyraEz/Liquid-Studio.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
